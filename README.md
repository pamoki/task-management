# タスク管理アプリ

## 概要
チームで利用できるタスク管理アプリを開発しました。
直感的に操作できるUIを意識して設計しています。

## 使用技術
- Flutter
- Dart

## 主な機能
- タスクの追加・削除
- タスクの状態管理（未完了 / 完了）

## 担当箇所
- UI設計
- 画面実装
- 一部ロジック開発

## 工夫した点
- タスクの視認性を高めるために色分けを行った
- ユーザーが直感的に操作できるUIを意識した

## チーム開発での経験
- メンバーと役割分担を行いながら開発を進めた
- 意見の違いが出た際は話し合いで方向性を統一した

## 今後の改善点
- UIのさらなる改善
- 機能追加（通知機能など）
